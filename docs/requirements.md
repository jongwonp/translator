# 기능 요건 정의서

## 1. 프로젝트 개요

- **프로젝트명**: Translator (가칭)
- **목적**: 영상 매체(유튜브, 빌리빌리 등)의 외국어 음성을 스크립트로 변환하고, 학습 목적의 단어 추출 및 자막 대용 기능을 제공하는 웹 애플리케이션
- **대상 사용자**: 영상 기반 외국어 학습자, 외국어 영상의 자막이 필요한 사용자

---

## 2. 용어 정의

| 용어 | 정의 |
|------|------|
| 스크립트 | 영상의 음성을 텍스트로 변환한 결과물 (타임스탬프 포함) |
| 단어장 | 사용자가 저장한 어려운 단어 목록 |
| 원본 언어 | 영상에서 사용되는 언어 |
| 번역 언어 | 스크립트를 번역할 목표 언어 |
| 세그먼트 | 타임스탬프로 구분된 스크립트의 한 단위 (한 문장 또는 한 구절) |

---

## 3. 사용자 유형

### 사용자 A: 언어 학습자
- **목적**: 외국어 영상에서 모르는 단어를 추출하여 개인 단어장에 저장하고 학습
- **주요 언어**: 일본어, 영어 (추후 확장 가능)
- **사용 기능**: 스크립트 생성 + 번역 + 단어 추출 + 단어장 관리

### 사용자 B: 자막 대용 사용자
- **목적**: 자막이 제공되지 않는 외국어 영상의 내용을 이해하기 위해 스크립트를 자막처럼 활용
- **주요 언어**: 중국어 영상 → 한국어 번역
- **사용 기능**: 스크립트 생성 + 번역 + 영상 연동 (단어장 불필요)

---

## 4. 기능 요건

### 4.1 핵심 기능

#### F-001: 영상 오디오 추출
- **설명**: 사용자가 입력한 영상 URL에서 오디오를 추출한다
- **입력**: 영상 URL (유튜브, 빌리빌리 등)
- **출력**: 오디오 파일
- **조건/제약**:
  - 지원 플랫폼: 유튜브, 빌리빌리 (yt-dlp가 지원하는 플랫폼)
  - DRM이 걸린 플랫폼(넷플릭스 등)은 지원 불가
  - 로컬 영상/오디오 파일 업로드도 지원

#### F-002: 음성 인식 (STT)
- **설명**: 추출된 오디오를 텍스트 스크립트로 변환한다
- **입력**: 오디오 파일 + 원본 언어 설정
- **출력**: 타임스탬프가 포함된 세그먼트 단위 스크립트
- **조건/제약**:
  - 다국어 지원: 일본어, 영어, 중국어 등
  - 각 세그먼트에 시작/종료 시간 포함
  - 사용자가 원본 언어를 직접 선택하거나 자동 감지

#### F-003: 스크립트 번역
- **설명**: 생성된 원본 스크립트를 사용자가 선택한 언어로 번역한다
- **입력**: 원본 스크립트 + 번역 목표 언어
- **출력**: 원문과 번역문이 나란히 표시되는 스크립트
- **조건/제약**:
  - 세그먼트 단위로 번역 (타임스탬프 유지)
  - 번역이 필요 없는 경우 건너뛸 수 있음 (원본 언어 = 학습 대상 언어인 경우)

#### F-004: 영상-스크립트 연동
- **설명**: 영상 플레이어와 스크립트를 연동하여 자막처럼 활용할 수 있도록 한다
- **입력**: 영상 URL + 타임스탬프 포함 스크립트
- **출력**: 영상 플레이어 + 연동된 스크립트 뷰
- **세부 기능**:
  - 스크립트의 특정 세그먼트를 클릭하면 영상이 해당 시점으로 이동
  - 영상 재생 중 현재 구간의 스크립트가 자동 하이라이트
  - 영상 재생에 따라 스크립트가 자동 스크롤
- **조건/제약**:
  - 유튜브: iframe API를 통한 완전 연동
  - 빌리빌리: 임베드 가능 시 연동, 불가 시 해당 시점 URL을 새 탭으로 열기

#### F-005: 회원가입/로그인
- **설명**: 사용자별 개인 데이터를 관리하기 위한 인증 시스템
- **입력**: 이메일(또는 아이디) + 비밀번호
- **출력**: 인증 토큰 (JWT)
- **조건/제약**:
  - 소규모 사용자 대상이므로 간단한 JWT 기반 인증

---

### 4.2 부가 기능 (단어장 관련 - 선택적 사용)

#### F-101: 단어 추출
- **설명**: 생성된 스크립트에서 사용자의 수준에 맞는 어려운 단어를 추출한다
- **입력**: 스크립트 (전체 또는 사용자가 선택한 구간) + 사용자 학습 수준
- **출력**: 단어 목록 (단어, 발음, 뜻, 원문 예문)
- **조건/제약**:
  - 스크립트 생성(F-002, F-003) 완료 후 사용자가 선택적으로 실행
  - 스크립트 생성과 단어 추출은 반드시 분리된 단계로 동작
  - 사용자가 전체 스크립트 또는 특정 구간을 선택하여 추출 가능

#### F-102: 단어 수준 조절
- **설명**: 사용자의 외국어 수준에 따라 추출되는 단어의 난이도를 조절한다
- **입력**: 사용자 학습 수준 설정
- **수준 기준**:
  - 일본어: JLPT 기준 (N5~N1)
  - 영어: CEFR 또는 빈도 기준 (초급/중급/고급)
  - 중국어: HSK 기준 (1~6급)
- **추출 로직 (하이브리드 방식)**:
  1. 빈도 리스트 기반 1차 필터링: 사용자 수준 이하의 기본 단어 제거
  2. 개인 단어장 기반 중복 제거: 이미 저장된 단어 제외
  3. LLM 기반 2차 분석: 후보 단어의 문맥 확인, 관용 표현 보충, 오탐 제거

#### F-103: 단어장 관리
- **설명**: 추출된 단어를 개인 단어장에 저장하고 관리한다
- **세부 기능**:
  - 추출된 단어 목록에서 저장할 단어를 선택
  - 단어장 조회 (언어별, 날짜별, 출처 영상별)
  - 단어 삭제
  - 단어 검색
- **저장 항목**: 단어, 발음, 뜻, 예문 (스크립트에서 추출), 출처 (영상 제목/URL), 저장 일시

#### F-104: 단어장 내보내기
- **설명**: 단어장 데이터를 외부 형식으로 내보낸다
- **출력 형식**: CSV (Anki 호환)

## 5. 비기능 요건

### 성능
- 10분 이내 영상의 스크립트 생성이 합리적인 시간 내에 완료되어야 함
- 동시 사용자 수: 2~5명 수준

### 보안
- 사용자 비밀번호 해싱 저장
- JWT 토큰 기반 인증
- 개인 단어장은 본인만 접근 가능

### 지원 환경
- 웹 브라우저 (PC/모바일 반응형)
- 최신 Chrome, Firefox, Safari 지원

---

## 6. 향후 확장 (당장 구현하지 않음)

- 단어장 공유: 친구 간 단어장 열람/공유 기능
- 학습 진도 추적: 단어 암기 여부, 복습 주기 관리
- 추가 언어 지원: 스페인어, 프랑스어 등
- 추가 플랫폼 지원: 기타 yt-dlp 지원 플랫폼
- 실시간 오디오 캡처: 시스템 오디오를 실시간으로 인식
